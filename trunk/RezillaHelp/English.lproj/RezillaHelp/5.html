Inspector window

The Inspector window displays information about a particular resource: type, ID, name and attributes. You can modify the ID or the name of a resource and the value of its different attributes. Note that none of the changes will be definitive until you save the resource map.

If the Inspector window is visible, you can just option-click (i-e click with the option key pressed down) on a resource in a resource map for the corresponding info to be displayed in the Inspector.
