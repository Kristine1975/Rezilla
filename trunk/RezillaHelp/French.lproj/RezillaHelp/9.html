Menu Edit

    Copy / Cut / Paste
    Les fonctions de Copier, Couper et Coller peuvent avoir différentes significations selon le contexte. Voir la section Copier, Couper et Coller plus loin.

    Clear
    Détruit la sélection. Cette fonction peut avoir différentes significations selon le contexte. Si une table de ressources est active et que des ressources y sont sélectionnées, elles sont supprimées de la table; si une fenêtre d'éditeur hexadécimal est active et que du texte est sélectionné dans un des cadres, cette sélection est détruite.

    Select all
    Cette commande permet de tout sélectionner: toutes les ressources dans une table de ressource, tout le texte dans une fenêtre de texte etc.

    Find...
    Dans l'éditeur hexadécimal, cette commande présente un dialogue dans lequel on peut entrer une chaîne de caractères à rechercher. Deux boutons-radios permettent de spécifier dans quel panneau la recherche doit s'opérer. Si l'on recherche dans le panneau de droite, une case à cocher permet en plus de spécifier si la recherche doit tenir compte de la casse des caractères.

    Find again
    Cette commande recherche, depuis la position courante, la chaîne entrée dans le dialogue de recherche (appelé par la commande Find). Elle utilise les mêmes critères que ceux qui ont été sélectionnés dans ce dialogue. Si le dialogue Find contient une chaîne vide, la commande est désactivée.

    Compare maps...
    Cette commande permet de comparer deux tables de ressources. Cela peut être utile si l'on veut voir les changements entre deux versions successives d'une application ou d'un fichier. Le dialogue permettant de sélectionner les deux fichiers dont on veut comparer les ressources comporte aussi quelques cases à cocher grâce auxquelles on peut ajouter des options de comparaisons: il est possible d'ignorer les différences relatives au nom des ressources, à leurs attributs et à leurs données binaires.

    Les résultats sont affichés dans une fenêtre comportant trois listes: la liste de gauche contient les ressources trouvées dans la première table de ressources et absentes de la seconde, la liste de droite contient les ressources trouvées dans la seconde table de ressources et absentes de la première. La liste du milieu contient les ressources communes aux deux tables de ressources et qui diffèrent selon les critères de comparaison. En cliquant sur une ressource dans cette liste, les données hexadécimales de la ressource sont affichées dans deux panneaux parallèles au bas de la fenêtre afin de les comparer: à gauche, les données provenant de la première table de ressources, et à droite, les données provenant de la deuxième. Les données peuvent être visualisées ou bien en format hexadécimal, ou bien en format texte au moyen des deux boutons-radios au bas de la fenêtre.
