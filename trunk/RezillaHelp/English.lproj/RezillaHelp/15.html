Version History

This project has been started in May 2003 and installed on SourceForge in June 2003: it is a work in progress, not all features are available yet. See the file Changes_Rezilla for details about the development of the various functionalities.
  • 2003-06-01 - 1.0.1a0, a1... - Successive alpha stages of the program.
  • 2004-05-07 - 1.0.1a - First public alpha release. Basic features.
  • 2004-05-24 - 1.0.2a - Second public alpha release. Preferences.
  • 2004-06-09 - 1.0.3a - Third public alpha release. Resizable windows. Style preferences.
  • 2004-11-09 - 1.0.4a - Fourth public alpha release. Templates editor. Many new preferences. Text resources editor.
