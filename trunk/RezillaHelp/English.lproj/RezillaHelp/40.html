Editors panel

In the Editors panel, one can set properties concerning specific editors.

Bitmap editor preferences

The Use 8-bit colors preference concerns the depth of color icons. If it is on, it forces to use a color depth of 8 rather than 32.

The Use Full Color Tables tells the Bitmap editor to use palettes with 256 colors. Otherwise the classical Apple's palette with only 32 colors is used.

Unicode editor preferences

If the Do Font Substitution preference is checked, Rezilla will automatically display Unicode text using the right fonts corresponding to the scripts used in the resource.

The Insert Byte Order Mark preference tells whether a byte order mark (aka BOM or code-point U+FEFF) should be inserted at the beginning of the resource. This Unicode symbol is often used to identify Unicode data in the UTF-16 encoding form, which is the form used by Rezilla.
