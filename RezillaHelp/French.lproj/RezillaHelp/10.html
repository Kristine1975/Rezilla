Menu Ressource

    New resource
    Permet de créer une nouvelle ressource dans la table de ressources courante. Une fenêtre de dialogue permet de spécifier le type, le numéro ID, le nom et les attributs de la nouvelle ressource. Une ressource est alors créée ne contenant aucune donnée. Noter que cette ressource est créée en mémoire et qu'il faudra sauvegarder le fichier pour la rendre définitive.

    Edit
    Appelle un éditeur pour la ou les ressources sélectionnées dans la table de ressources courante. Si Rezilla ne trouve aucun éditeur interne dédié pour une ressource particulière, il cherche un plugin capable de l'éditer (les plugins ont été introduits dans la version 1.1 de Rezilla). Si aucun ne convient,Rezilla cherche alors si un gabarit est disponible afin d'invoquer l'éditeur de gabarits. En cas d'échec, il appelle l'éditeur hexadécimal. Cette commande équivaut à double-cliquer sur les ressources dans la table.

    Template Editor
    Appelle l'éditeur de gabarits pour la ou les ressources sélectionnées dans la table de ressources courante. Si Rezilla ne trouve aucun gabarit, il appelle l'éditeur hexadécimal. Cette commande équivaut à double-cliquer sur les ressources dans la table avec la touche Majuscule enfoncée.

    Hexadecimal Editor
    Appelle l'éditeur hexadécimal pour la ou les ressources sélectionnées dans la table de ressources courante. Cette commande équivaut à double-cliquer sur les ressources dans la table avec la touche Option enfoncée.

    Edit as Type...
    Force Rezilla à utiliser un type de remplaceement. Dans le dialogue affiché, on peut spécifier le type soit à partir d'un menu, soit en le tapant directement dans le champ de saisie.

    Edit with Plugin...
    Force Rezilla à éditer une ressource en utilisant un éditeur externe défini sous forme de plugin s'il en existe un qui convient. Cette copmmande est équivalent au raccourci-clavier option-contrôle-commande-E.

    Remove
    Supprime toutes les ressources sélectionnées de la table de ressources.
    touche Option enfoncée.

    Duplicate
    Duplique la ou les ressources sélectionnées dans la table de ressources. Cette commande attribue un numéro unique d'identification (ID) aux ressources dupliquées. On peut modifer ces valeurs par la suite dans la fenêtre Inspecteur. L'algorithme calculant un identificateur unique renvoie la première valeur disponible au-dessus de la valeur courante.

    Get Info
    Appelle la fenêtre Inspecteur pour le première ressource sélectionnée dans la table de ressources courante. C'est équivalent à cliquer sur une ressource dans une table de ressources en maintenant la touche Option enfoncée.
