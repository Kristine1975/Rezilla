The MENU editor

The menu editor lets you edit standard resources of type MENU, that is to say resources which use the standard format for this type. If the resource uses a custom format defined in a Menu Definition resource (MDEF), it can't be edited with this editor. The menu editor looks for an MDEF resource and warns the user. On the other hand, it supports extended information stored in resources of type xmnu.

While editing a MENU resource, the editor checks for the presence of an xmnu resource with the same ID and, if it finds one, reads its contents. The interface displays global information concerning the menu itself at the top of the window and the data concerning each of its items in the right panel of the window. Standard item data are located in the top half of this panel (down to the style popup) while extended data (xmnu) are in the lower half (starting from the modifiers).

If no xmnu resource was detected, the editor displays default values in the corresponding fields: if any of these values is modified, the editor will propose to create an xmnu resource and save the extra information therein.

Note that xmnu resources can also be edited directly using the template editor.
