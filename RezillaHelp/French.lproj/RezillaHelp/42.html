Export des tables de ressources

La fonction d'export de Rezilla permet de générer des fichiers en divers formats contenant toutes les données relatives à une table de ressources. Les formats disponibles sont:
  • Texte
  • HTML
  • XML
  • DeRez

En formats XML et texte, il est possible de décider d'inclure les données binaires de chaque ressource ou pas: voir la section Préférences. Celles-ci peuvent être représentées soit en encodage Base64 (qui est le format utilisé sous Mac OSX pour de nombreux types de fichiers XML tels que les fichiers de préférences ou les fichiers de listes de propriétés), soit sous forme hexadécimale (comme dans un éditeur hexadécimal). Ce choix peut être spécifié dans les Préférences également.

Pour des exports en format XML, il y a deux façons de représenter les données basées sur deux DTD (Définitions de Type de Document) différentes. La première représente chaque propriété au moyen d'une clé spécifique tandis que l'autre les représente comme attributs de certains éléments (élément type et élément ressource).

			<Resource>
				<ResourceID>204</ResourceID>
				<ResourceName>Low Memory Warning</ResourceName>

				<ResourceSize>14</ResourceSize>
				<ResourceData>
				AGgAggDNAegAzEREMAo=
				</ResourceData>
			</Resource>
		</ResourcesArray>
	</Type>
	<Type>
		<TypeCode>DITL</TypeCode>
		<ResourcesArray>
			<Resource>
				<ResourceID>204</ResourceID>
				<ResourceName></ResourceName>
				<ResourceFlags>20</ResourceFlags>
				<ResourceSize>120</ResourceSize>
				<ResourceData>
				AAEAAAAAAEcBIABbAVwEAk9LAAAAAAAKAEsAOgFciFdNZW1v

				ZG9jdW1lbnRzLiDl

		</ResourcesArray>
	</Type>
