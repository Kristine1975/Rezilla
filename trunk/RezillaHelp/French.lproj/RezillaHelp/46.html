Éditer un gabarit

Bonne nouvelle ! Il existe un gabarit pour les ressources de type TMPL. Cela signifie que l'on peut éditer ces ressources au moyen de... l'éditeur de gabarits ! Celui-ci présente une liste de paires balise/étiquette sous forme de chaînes de caractères, et permet d'ajouter ou de retirer certaines paires, de les modifier etc. La seule chose à savoir est quelle balise on doit utiliser pour chaque type de données. La section suivante répond à cette question.

Maintenant que la structure des gabarits a été expliquée, il est possible de décrire avec davantage de précision le format d'une ressource TMPL. Il s'agit d'une liste de paires comportant un code de quatre caractères et une chaîne de Pascal ; en d'autres termes, c'est une série de données de type TNAM et PSTR alternativement.
