Panneau Compare

Dans le panneau Compare, on peut fixer des valeurs par défaut pour les critères de comparaison apparaissant dans le dialogue Compare Maps: ces valeurs peuvent être modifiées par la suite dans le dialogue lui-même. On peut aussi fixer une valeur par défaut pour les boutons radios apparaissant au bas de la fenêtre Compare Results affichant les résultats d'une comparaison de tables de ressources et indiquant le format à utiliser pour afficher les données binaires des ressources (hexadécimal ou texte): ces valeurs peuvent êtremodifiées par la suite lorsque la fenêtre est affichée.
