Menu Fichier

    New Map
    Permet de créer un nouveau fichier de ressources. Un dialogue standard permet d'indiquer le nom et l'emplacement du nouveau fichier de ressources à créer; deux boutons-radios permettent de spécifier si la table de ressources doit être stockée dans la branche de données ou dans la branche de ressources du fichier.

    Open...
    Présente un dialogue standard pour ouvrir une table de ressources. Dans ce dialogue, trois boutons-radios permettent de spécifier la branche dans laquelle rechercher les ressources. Les options proposées sont:
    • "any fork": dans ce cas, Rezilla cherche en premier une table de ressources dans la branche de données. S'il n'en trouve aucune, il cherche alors dans la branche de ressources.
    • "data fork": recherche une table de ressources seulement dans la branche de données du fichier
    • "resource fork": recherche une table de ressources seulement dans la branche de ressources du fichier
    On peut aussi sélectionner la case à cocher Read Only: si elle est cochée, la table sera ouverte en mode lecture seule et toutes les commandes d'édition seront désactivées.

    Recent items
    Ce sous-menu contient les noms des dix plus récentes tables de ressources ouvertes dans Rezilla. Si on sélectionne un article de ce menu, la table de ressources correspondante est ouverte.

    Close
    Ferme la fenêtre se trouvant au premier-plan.

    Revert
    Cette commande permet de restaurer la dernière version sauvegardée sur disque dans le cas où la table courante a été modifiée et que l'on ne veut pas conserver ces modifications.

    Save
    Sauvegarde les modifications sur disque.

    Save As...
    Présente un dialogue standard pour spécifier un fichier dans lequel stocker la table de ressources courante. Trois boutons-radios permettent de spécifier dans quelle branche stocker la table. Les options proposées sont:
    • "same fork": utiliser la même branche que la table courante.
    • "data fork": sauvegarder dans la branche de données.
    • "resource fork": sauvegarder dans la branche de ressources.

    Import...
    Cette commande a différentes significations selon le contexte:
    • si la fenêtre active appartient à l'éditeur de ressources 'icns', la commande Import permet d'importer les données d'une famille d'icones depuis un fichier ayant l'extension .icns.

    Export...
    Cette commande a différentes significations selon le contexte:
    • si la fenêtre active est une table de ressources, la commande Export permet d'exporter la structure et le contenu d'une table de ressources dans des fichiers selon divers formats. Les formats disponibles sont:
      • Texte
      • HTML
      • XML
      • DeRez
      Pour plus d'informations, voir la section Export et import des tables de ressources.

    • si la fenêtre active appartient à l'éditeur de ressources 'icns', la commande Export permet d'exporter la ressource courante dans un fichier ayant l'extension .icns.

    Plugins...
    Cette commande affiche une fenêtre de dialogue pour obtenir des informations concernant les plugins et réarranger leur ordre de préférence, type par type.
