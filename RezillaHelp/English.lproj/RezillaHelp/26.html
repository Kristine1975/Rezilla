The list editor

The string lists editor lets you edit resources of type STR#, MBAR, Mcmd, RidL which represent a counted list of textual or numerical values:

  1. STR# resources contain strings of characters. Each string can have as much as 255 characters.
  2. MBAR resources contain decimal values stored on two bytes
  3. Mcmd and RidL resources contain decimal values stored on four bytes

The items of the list are numbered by the editor.

One can select a list item just by clicking in the part containing the numerotation, at the left of the edit field. The selected element is enclosed in a red frame. To unselect it, just click a second time in the same area. Several elements can be thus selected.

One can cycle through the edit fields using the Tab key. If the Shift key is pressed simultaneously, rotation goes backwards.

A string lists editor window has two buttons to add or remove an element in the list:

  1. the Plus button inserts a new element just before the first selected element of the list or at the end of the list if no element is currently selected.

    Tip: if you press the Option key while clicking on the Plus button, a dialog is displayed asking you how many items you want to add.

  2. the Minus button deletes all the selected elements. If no element is selected then the last element of the list will be deleted.

The string lists editor supports drag-and-drop. Click on the left part of an item and move the mouse inside the list without releasing its button in order to move this item: the element is dropped at the location where the mouse button is released. If the Option key is pressed simultaneously, the element is duplicated instead of being just moved.

One can also drag and drop an element onto a window belonging to another application: the string is inserted in the target window provided the other application supports drag-and-drop of text elements. For instance, dragging and dropping an element on the Finder results in a text clipping being created there. On the contrary, if one drops, on a string lists editor window, a piece of text coming from another application, a new element is created in the list with this text (truncated to a maximum of 255 characters).

Note that STR#, MBAR, Mcmd, RidL resources can also be edited using the templates editor.
