Export panel

In the Export panel, one can choose the encoding used to store the binary data: Base64 encoding or hexadecimal encoding. The Include binary data in output checkbox lets specify whether the data for each resource should be included in the export file: this currently applies only to Text and XML exports, not HTML or DeRez. For more info, see the Exporting/importing a resource map section.

In the Text editor signature field, you can specify the creator type of a text editor which will be attributed to the output file so that you can later edit it with this editor by a double-click.
