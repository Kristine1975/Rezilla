Template Editor

Rezilla is able to parse resources using descriptive templates and to build a graphical interface out of it, allowing to easily edit all the elements of the resource. For instance, the following image shows how Rezilla would edit a Txtr resource containing font and size information for a document.

There are three kinds of locations where TMPL resources can be found. They are described in details in the following subsections:

  • internal templates are predefined templates stored inside the application. Rezilla ships with more than 360 predefined templates covering the most currently used resource types;
  • external templates are additional templates that one can add to one of the Application Support folders;
  • local templates are TMPL resources present in the currently edited resource map.

When Rezilla attempts to edit a resource using a template, it looks for the appropriate template in all these locations in the following order: local, external, internal. You can thus override the definition of an internal template by putting a new definition in the external locations.

Templates editor windows have three buttons:

  • the Cancel button lets you abort the editing and close the window without asking if modifications should be saved.
  • the Validate button lets you save your modifications and close the window.
  • the Revert button lets you discard the recent modifications and revert to the last saved version of the resource in the resource map in memory (not the last saved version of the resource on disk).

Note that pressing the Enter key (not the Return key) is equivalent to clicking on the Validate button.

If the Close button (the leftmost red little button in the title bar) is clicked and if any of the resource fields has been edited, Rezilla will ask you if you want to save your modifications. If the answer is yes, the modified resource is installed in the resource map. This does not mean though that the new data are saved to the disk: this happens only when you close the resource map table itself or invoke the Save menu item when this resource map table is at the front.
