Glisser-Déposer

Rezilla supporte le Glisser-Déposer (drag and drop). On peut faire glisser une sélection de ressources d'une table de ressources à une autre afin de les copier dans la table de destination. Si la table de destination est la même que celle de départ, les ressources sont simplement dupliquées.

Lorsque l'on fait glisser une sélection de ressources provenant d'une table de ressources jusqu'à la corbeille qui se trouve sur le Bureau, ces ressources sont supprimées de la table.

Si l'on fait glisser un morceau de texte depuis un éditeur de textes externe sur une table de ressources de Rezilla, une nouvelle ressource de type TEXT contenant ce texte est créée (la taille du texte est limitée à 32k). Plus généralement, si l'on fait glisser une sélection d'un certain type (flavor) sur une table de ressources dans Rezilla, des ressources le même type sont créées. Par exemple, si l'on fait glisser une image sélectionnée dans un programme tel que GraphicConverter jusqu'à Rezilla, deux ressources sont créées: une ressource de type Rect contenant les dimensions de l'image, une autre ressource de type PICT contenant une représentation au format PICT de cette image.
