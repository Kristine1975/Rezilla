Export

La fonction d'export de Rezilla permet de générer un fichier XML contenant toutes les données relatives à une table de ressources. Le code binaire de chaque ressource est encodé en encodage Base64 (qui est le format utilisé sur MacOSX pour divers fichiers XML comportant des données binaires, comme les fichiers de préférences, les fichiers de listes de propriétés etc.).
