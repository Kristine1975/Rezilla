Tables de ressources

À l'ouverture d'un fichier de ressources, la table de ressources est affichée dans une fenêtre donnant la liste des divers types de ressources contenus dans le fichier: si l'on clique sur le petit triangle placé à la gauche du type, la table affiche, au second niveau, la liste des différentes ressources de ce type, avec leur numéro (ID), leur taille et leur nom (si elles en ont un).

Le bas de la table de ressources comporte des indicateurs affichant le nombre de types différents contenus dans cette table et le nombre total de ressources. Les lettres DF ou RF à droite du nombre de ressources indiquent si la table de ressources a été ouverte à partir de la branche de données (DF signifie data fork) ou de la branche de ressources du fichier (RF signifie resource fork).

Le coin inférieur gauche de la fenêtre affiche une petite icone représentant un verrou: si le verrou est fermé, la table de ressources est en mode lecture seule. Dans ce mode, on ne peut exécuter aucune commande d'édition: toutes les commandes de menu relatives aux fonctions d'édition de la table de ressources sont désactivées. On peut cependant visualiser le contenu d'une ressource au moyen de l'éditeur hexadécimal mais il sera en mode lecture seule lui aussi.
