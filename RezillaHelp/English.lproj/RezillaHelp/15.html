Version History

This project has been installed on SourceForge in June 2003: it is a work in progress, not all features are available yet. See the file Changes_Rezilla for details about the development cycle.
  • 2004-05-07 - 1.0.1a - First public alpha release. Basic features.
  • 2004-05-24 - 1.0.2a - Second public alpha release. Preferences.
  • 2004-06-09 - 1.0.3a - Third public alpha release. Resizable windows. Style preferences.
  • 2004-11-22 - 1.0.4a - Fourth public alpha release. Templates editor. 'Text' resources editor. New preferences.
  • 2005-02-22 - 1.0.5b - First public beta release. Bitmpap editor, Unicode editor, Aete editor, Pict viewer, Sound player, new preferences, new templates.
