Préférences

Rezilla permet de fixer diverses préférences. On accède aux préférences au moyen de la commande Préférences dans le menu de l'application (menu Rezilla) ou au moyen du raccourci clavier Command-, (taper la touche virgule en maintenant la touche commande enfoncée). Un dialogue multipage s'affiche comportant trois panneaux:
  • le panneau General
  • le panneau Export
  • le panneau Compare

Dans le panneau General, on peut fixer le nombre maximal d'articles contenus dans le menu Recent Items (par défaut 10). Un bouton Reset permet d'effacer la liste actuelle. On peut aussi spécifier la branche qui sera utilisée par défaut pour créer une nouvelle table de ressources (cela correspond aux boutons radios affichés dans le dialogue New Map: ce choix par défaut peut être modifié plus tard dans le dialogue).

Dans le panneau Export, on peut choisir la DTD à utiliser pour créer un export XML (voir à la section Export dans les fichiers d'aide). On peut également spécifier l'encodage à utiliser pour les données binaires: encodage Base64 ou représentation hexadécimale. La case à cocher Include binary data in output permet de spécifier si les données binaires de chaque ressource doivent être inclues dans les fichiers d'export: cela concerne pour le moment uniquement les formats Texte et XML, pas HTML.

Dans le panneau Compare, on peut fixer des valeurs par défaut pour les critères de comparaison apparaissant dans le dialogue Compare Maps: ces valeurs peuvent être modifiées par la suite dans le dialogue lui-même. On peut aussi fixer une valeur par défaut pour les boutons radios apparaissant au bas de la fenêtre Compare Results affichant les résultats d'une comparaison de tables de ressources et indiquant le format à utiliser pour afficher les données binaires des ressources (hexadécimal ou texte): ces valeurs peuvent êtremodifiées par la suite lorsque la fenêtre est affichée.
