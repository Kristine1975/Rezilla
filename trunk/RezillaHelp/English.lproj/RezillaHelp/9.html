Exporting

The exporting functionality lets you generate an XML file containing all the data of the resource map (all resources of all types, with their ID, name, attributes etc.). The binary code for each resource is encoded in Base64 encoding (which is the format used on MacOSX for many other types of binary data stored in XML files, like for instance preferences files, property lists etc.).
