Panneau Export

Dans le panneau Export, on peut choisir la DTD à utiliser pour créer un export XML (voir à la section Export dans les fichiers d'aide). On peut également spécifier l'encodage à utiliser pour les données binaires: encodage Base64 ou représentation hexadécimale. La case à cocher Include binary data in output permet de spécifier si les données binaires de chaque ressource doivent être inclues dans les fichiers d'export: cela concerne pour le moment uniquement les formats Texte et XML, pas HTML.

Dans le champ appelé Text editor signature, on peut indiquer le créateur d'un éditeur de textes à attribuer aux fichiers produits afin que, par la suite, ceux-ci soient automatiquement ouverts dans cet éditeur lorsqu'on fait un double-clic sur leur icone.
