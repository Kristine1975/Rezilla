Lists of elements in templates

Some resources contain lists of data, for instance a list of strings, a list of graphical objects etc. Templates are able to handle various kinds of lists: as a header of each list, the Templates editor displays a count of the items currently in the list and two buttons with a Minus and a Plus sign respectively. Typically you'll use the Minus button to remove an item from the list and the Plus button to add a new item.

A list item can contain different fields: they are all displayed in a common frame which it is very easy to identify. Clicking on one such item selects it, which the editor displays by drawing a red border around it.

When clicking on the Minus button:

  • if an item of this list is selected, this item will be removed from the list.
  • if no item is selected, the last item of the list will be removed.
  • if the list is empty, nothing happens.

When clicking on the Plus button:

  • if an item of this list is selected, a new item is inserted before the selected item.
  • if no item is selected, a new item is added at the end of the list.
