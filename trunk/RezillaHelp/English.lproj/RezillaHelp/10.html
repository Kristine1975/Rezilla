Resource menu

    New Resource
    Create a new resource in the current resource map. A dialog window lets you specify the type, ID, name and attributes of the new resource. A resource is then created with empty data: the resource is created in memory and you will have to save your file to make it permanent.

    Edit
    Calls an editor for all the resources selected in the resource map table. This is equivalent to double-clicking on the resources in the table. If no Interface editor is available for a particular resource: if no such editor is found, then it looks for a plugin able to edit it (plugins have been introduced in version 1.1 of Rezilla), In case of failure, Rezilla will look for a template corresponding to the resource's type and invoke the Template Editor therewith. It no template is found, the last resort will be to call the Hexadecimal Editor.

    Template Editor
    Calls the Template Editor for all the resources selected in the resource map table. This is equivalent to double-clicking on the resources with the shift key down. It no template is found, Rezilla will invoke the Hexadecimal Editor.

    Hexadecimal Editor
    Calls the Hexadecimal Editor directly for all the resources selected in the resource map table. This is equivalent to double-clicking on the resources with the option key down.

    Edit as Type...
    Forces Rezilla to edit a particular resource using another type. A dialog is displayed in which one can select the type to use instead of the original type of the resource.

    Edit with Plugin...
    Forces Rezilla to edit a particular resource using a third party editor defined via a plugin if there is one available. This is equivalent to the option-control-command-E keyboard shortcut.

    Remove
    Removes all the selected resources from the resource map table. Note that this action won't be definitive until you save explicitely the file.

    Duplicate
    Duplicates all the selected resources in the resource map table. This command attributes an unique ID for the duplicated resources. You can change these values later with the Inspector window. The algorithm calculating a unique ID attempts to find the first available value above the current value.

    Get Info
    Calls the Inspector window for the first selected resource in the resource map table. This is equivalent to option-clicking on a resource in a resource map.
