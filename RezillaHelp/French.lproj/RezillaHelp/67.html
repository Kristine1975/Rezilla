Historique

Ce projet a été installé sur le site de SourceForge en juin 2003. Voir le fichier Changes_Rezilla pour plus de détails techniques concernant l'évolution du projet.
  • 2004-05-07 - 1.0.1a - Première version alpha publique du programme. Fonctions de base.
  • 2004-05-24 - 1.0.2a - Deuxième version alpha publique du programme. Préférences.
  • 2004-06-09 - 1.0.3a - Troisième version alpha publique du programme. Fenêtres redimensionnables. Préférences de style.
  • 2004-11-22 - 1.0.4a - Quatrième version alpha publique du programme. Éditeur de gabarits. Éditeur de ressources texte. Nouvelles préférences.
  • 2005-02-28 - 1.0.5b - Première version beta publique du programme. Éditeur d'icones, éditeur Aete, éditeur Unicode, visualisateur de ressources PICT, exécution de ressources son, nouvelles préférences, nouveaux gabarits.
  • 2005-06-16 - 1.0.6b - Seconde version beta publique du programme. Éditeur de menus. Support AppleScript complet. Nouvelles balises de gabarit pour menus de valeurs cumulatives.
  • 2005-09-16 - 1.0.7 - Première version finale du programme. Ajoute un éditeur de listes, des fichiers projet pour XCode 1.5 et 2.1, et quelques corrections mineures.
  • 2006-02-10 - 1.0.8 - Nouveau format pour les balises RECT. Améliorations de l'interface utilisateur (media en lecture seule, valeurs hexadécimales). Résolution de bogues.
  • 2006-10-15 - 1.1 - Plugins, sélectionneurs de resources, éditeur icns, éditeur d'images. Import/export. Nombreuses améliorations et résolutions de bogues.
