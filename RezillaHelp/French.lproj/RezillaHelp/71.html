Téléchargement

Rezilla est un projet Open Source (logiciel libre). Son code source est public et peut être obtenu sur le site de SourceForge à l'adresse suivante:

http://sourceforge.net/projects/rezilla

Les versions compilées de Rezilla sont disponibles à

http://sourceforge.net/project/showfiles.php?group_id=83267

ou sur ma page Web à

http://perso.orange.fr/bdesgraupes/rezilla.html

Le code est sous contrôle CVS (Concurrent Version System). On peut obtenir la version la plus récente des fichiers sources au moyen d'un programme client CVS. Voir les instructions figurant à:

http://sourceforge.net/cvs/?group_id=83267

On peut aussi naviguer en ligne dans la hiérarchie des fichiers sources à l'adresse suivante:

http://cvs.sourceforge.net/cgi-bin/viewcvs.cgi/rezilla
