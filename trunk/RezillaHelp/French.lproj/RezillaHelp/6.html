Modificateurs

Lorsque l'on fait un double-clic sur une ressource dans la table, on peut aussi enfoncer simultanément une touche modificatrice:

  • si la touche Majuscule est enfoncée pendant un double-clic, l'éditeur de gabarits est appelé directement pour cette ressource. Si aucun gabarit n'est disponible, l'éditeur hexadécimal est appelé à la place.
  • si la touche Option est enfoncée pendant un double-clic, l'éditeur hexadécimal est appelé directement pour cette ressource.

Plusieurs ressources peuvent être sélectionnées simultanément dans la table afin de les éditer.

En cliquant (simple clic) sur une ressource avec la touche Contrôle enfoncée, on appelle la fenêtre de l'Inspecteur pour cette ressource. Cela équivaut à selectionner la ressource et choisir le commande Get Info dans le Menu Ressource. Voir la section fenêtre de l'Inspecteur plus loin.

Noter que cliquer sur un des triangles avec la touche Option enfoncée permet d'agir sur tous les types de ressources à la fois: si le triangle était fermé, tous les types de ressources seront ouverts, et inversement si il était ouvert alors tous les types de ressources seront fermés.
