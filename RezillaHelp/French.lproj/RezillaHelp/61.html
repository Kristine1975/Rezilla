Problèmes

Il est possible de contacter l'auteur pour toutes sortes de questions, problèmes rencontrés etc. à l'adresse suivante:

bdesgraupes@users.sourceforge.net

La page Web que voici comporte les informations les plus récentes concernant Rezilla:

http://webperso.easyconnect.fr/bdesgraupes/rezilla.html

Il existe aussi une liste de discussion, rezilla-users, pour débattre de tous les sujets liés à l'utilisation de Rezilla, comment écrire de nouveaux gabarits de ressources, comment écrire des scripts en langage AppleScript pour piloter Rezilla, etc. Les nouvelles versions sont également annoncées sur cette liste. On peut s'inscrire ou se désinscrire à l'adresse suivante:

http://lists.sourceforge.net/mailman/listinfo/rezilla-users Pour envoyer un message à la liste, il suffit d'adresser un e-mail à rezilla-users@lists.sourceforge.net
