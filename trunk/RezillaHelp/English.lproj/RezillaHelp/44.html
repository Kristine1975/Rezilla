Templates Syntax

This section explains how to write a template for a particular resource. Templates are resources of type TMPL which use a very simple syntax: they are made of a list of label/tag pairs describing the type of a particular value in the resource. A template thus describes all the values in a particular resource: to edit it, the template editor reads the resource's data from left to right and compares it with the template.
